# Déroulement de l'expérience utilisateur (pas encore réalisé)

- l'utilisateur sélectionne sa langue, la longueur du text désiré, et clique sur un bouton pour commencer 
- l'application affiche un texte aléatoire
- l'utilisateur le lit
- l'utilisateur appuie sur la touche espace lorsqu'il a fini de lire
- le texte est remplacé par une page blanche
- l'utilisateur écrit ce dont il se souvient
- le texte original s'affiche à côté de ce que l'utilisateur a écrit
- l'utilisateur note les différences entre ce qu'il a écrit et le texte original 
- L'utilisateur se note

# Références et analyse

[papier sur l'auto évaluation](https://pubs.sciepub.com/education/3/4/10/)
[thread reddit sur un enseignant qui a instauré l'auto évaluation dans sa classe](https://www.reddit.com/r/Professors/comments/vd3nj1/what_happened_when_i_allowed_students_to_grade/)
[self assessment is more meaningful than self grading](https://www.researchgate.net/publication/50236003_Students_as_the_Definitive_Source_of_Formative_Assessment_Academic_Self-Assessment_and_the_Self-Regulation_of_Learning)

les principaux inconvénients de l'auto-évaluation semblent être:
- les étudiants plus jeunes se donnent une note trop élevée
- les étudiants ne savent pas quoi évaluer si on ne leur donne pas d'indication
- les étudiants ont tendance à survaloriser l'effort qu'ils ont fourni par rapport à la qualité

les principaux avantages de l'auto-évaluation semblent être:
- les étudiants la perçoivent comme plus juste
- les étudiants savent s'ils ont fourni un effort ou non
- les étudiants progressent en auto-évaluation avec le temps (leur évaluation se rapproche de celle du professeur)
- très rapide et simple à mettre en place
